# Professional E-commerce Platform

A full-featured, modern E-commerce web application built with **React** and integrated with **Firebase** for back-end services. This platform features a dynamic product catalog, secure user authentication, and a robust shopping cart system.

## ✨ Key Features

- **🛒 Integrated Shopping Cart**: Add, remove, and manage items with real-time updates.
- **🔐 User Authentication**: Secure Login and Signup functionality powered by Firebase.
- **🏷️ Product Categories**: Browse products specialized by categories (Men, Women, Kids).
- **📱 Responsive UI**: High-quality, responsive design using custom CSS for a premium aesthetic.
- **🔥 Dynamic Rendering**: Uses React hooks and context for efficient state management.

## 🛠️ Tech Stack

- **React**: Core library for building the user interface.
- **Firebase**: Backend-as-a-Service for authentication and data storage.
- **React Router**: For seamless single-page application navigation.
- **CSS3**: Custom styling and modern layouts.

## 📂 Project Structure

- **`/frontend`**: The main React application.
  - `src/Mycomponents`: Highly modular UI components (Navbar, Hero, Cart, etc.).
  - `src/Page`: Main application pages.
  - `src/Context`: Global state management.

## 🚀 Getting Started

To get the project up and running locally:

1. **Navigate to the frontend**:
   ```bash
   cd frontend
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Environment Setup**:
   Create a `.env` file in the `frontend` directory and add your Firebase configuration keys (see `src/Mycomponents/Firebase/FirebaseConf.js` for expected variables).
4. **Start the development server**:
   ```bash
   npm start
   ```
